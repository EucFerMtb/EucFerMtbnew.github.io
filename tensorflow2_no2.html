機械学習を行う為のツール導入について

5/28 2023

カテゴリー : 機械学習実装の手順2

このページでは、機械学習を進める上で必要なツールの導入を行います. 具体的には, tensorflow2やmatplotlibを入れていきます. 汎用性ある内容も記載しています. また, まだ機械学習の仮想環境構築ができていない場合は, 右のタスクバーの「機械学習を行う為の仮想環境構築について」 を参考に構築を行ってからこのページの内容を参照して頂けると良いかと思います.

5. tensorflow2のインストール

早速, 仮想環境内にtensorflow2を入れていきましょう. コマンドに"pip install tensorflow"を入力し, Enterしてください. しばらく成功するまで待ちましょう.

インストール成功したら, 検証します. コマンドに, "python -c "import tensorflow as tf;print(tf.reduce_sum(tf.random.normal([1000, 1000])))" と入力してEnterしてください. 最後, tf.Tensor(-674.89984, shape=(), dtype=float32)みたいなものが出たらOKです.

6. tensorflow2のアップデート

tensorflow2のインストールが確認できたら, コマンドに"pip install --upgrade tensorflow"と入力して, Enterしてください. これで, tensorflow2のアップデートができます.

7. matplotlibのインストール

ここでは, matplotlibのインストールの紹介にいきます. matplotlibは, グラフを描写するとき等に必要不可欠なものになるので入れておくと良いでしょう.
コマンドに, "pip install -U matplotlib"を入力して, Enterしてください.

最後に～割と重要なこと～

上記は, tensorflow2やmatplotlibのインストール方法を紹介しました. 今後, 例えば機械学習を進める(VSCode)上で必要なツールがある場合, コマンドで「ERROR表示で～～～をinstallせよ」的なものがでますので安心してください. そのときは, コマンドに"pip install ～～～"と書いてEnterすればインストールされます (tensorflow2やmatplotlibはその例).
また, インストールとアップデートを両方同時にやる方法として, コマンドに "pip install --upgrade ～～～" 或いは "pip install -U ～～～"と書いて実行すればいけます (Installing ～～～～～. Successfully installed～～～みたいなものが出たらOKです).
それ以外に起こりうりそうなことを纏めておきます.

  • no module 'sklearn'がでたら・・・コマンドで"pip install -U scikit-learn"を入力
  • cv2(open cvなどで利用)はpip3対応なので, "pip3 install --upgrade opencv-python"とする必要があります(padasも同様です).
