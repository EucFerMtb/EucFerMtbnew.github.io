機械学習を行う為の仮想環境構築について

5/27 2023

カテゴリー : 機械学習実装の手順1

このページでは、tensorflow2を使って、機械学習を進める方法を紹介します. よくいろんな参考書を見ると、Anacondaを使って機械学習を行うケースが多いですが、 ここでは、Anacondaを使わずにやっていこうと思います.
余談ですが、tensorflowはバージョンが1と2がありますが、2を勧めます. 理由は、ざっくりいえば2の方がより便利に&より簡単にコードを書ける&そのおかげでコードもより見やすいからです.

0. 実装するPCについて

今回は、
  • OS : Windows 11 Home 64bit
  • CPU : Intel Core i7-1255U
  • python : Python 3.11.1
でやっていきます(自分がMacに弱いことは・・・ゲホンゲホン).

1. 前提

前提として、Visual Studio Codeとpythonを使います.
Visual Studio Codeやpythonのインストール方法に関しては, インターネットで検索すると たくさん出てくるので, ここでは割愛します. ご自身で参考書をお持ちの方は、pythonのインストール方法が書いてあればそれでもよいと思います.
pythonがちゃんとインストールされたか確認してください. コマンドに"py -V"と入力して、 "Python 3.X.Y"(X,Yは任意の数)が出力されればOKです (※"pip3 --version"でpipのversionが分かります).

2. 機械学習用のフォルダーを作る(仮想環境構築)

インストールが終わったら、まずは機械学習を実装するためのフォルダー(仮想環境)を作っていきます. 任意の場所に任意の名前でフォルダーを作成してください. 以下は、「ドキュメント」フォルダー内に「tensorflow」というフォルダーを作成した例です.

3. 仮想環境作成ができる標準ツールの為のファイルの作成

では, 2で作成した「tensorflow」フォルダーを開いて, 検索タブのところを削除して"cmd"と入力します.

すると, コマンドプロンプトが表示されると思います.
続いて, コマンドに"py -m venv --system-site-packages .\env"を入力してEnterしましょう.
(余談ですが, このvenvというのはPyhtonの仮想環境作成ができる標準ツールです. これは仮想環境の作成を行うコマンドで, ".\env"はその環境を保存するフォルダ名になります. 基本的には, この".\env"フォルダー内で機械学習用のファイルを作成していくことになります. 詳しいことは省きますが, フォルダー名は.\envがよいです).

すると, 以下のようにtensorflowのフォルダー内に「env」というフォルダーが作成されます.

4. 仮想環境の有効化

仮想環境の有効化をします. コマンドに".\env\Scripts\activate"と入力してEnterしてください.

Enterすると, 以下のような(仮想環境)コマンドに切り替わります.

出来たら, コマンドに"py -m pip install update pip"を入力してEnterしましょう. 以下のような感じになったらOKです.

続いて, "py -m pip install --upgrade pip"を入力して, 以下のようにSuccessfully なんたら～とでたら大丈夫です.

これまた余談ですが, 仮想環境を後で終了する場合は, "deactivate"コマンドを実行します (ぶっちゃけいえば, ×印押して消しても大丈夫な気がしてます).

以上が, 機械学習を進める上で必要な仮想環境構築の手順になります. この次にtensorflow2や機械学習をするのに欠かせないツールのインストールに行きますが, それに関しては右のタスクバーの「機械学習を行う為のツール導入について」にとんでください.
